# 喃喃 · Murmur — 产品设计文档

## 一、产品定位

一款以聊天方式书写日记的桌面应用，主打**隐私安全**与**温暖体验**。支持文字、图片、Markdown 长文混合记录，内置 AI 总结与反馈功能。

- **产品名称**：喃喃自语（中文） / Murmur（英文）
- **目标平台**：macOS / Windows / Linux
- **商业模式**：一次性买断（Mac App Store / Microsoft Store / 独立分发 + License Key）
- **技术框架**：Tauri v2（Rust 后端 + Web 前端）

## 二、核心特性

### 2.1 双空间隐私设计

- **私密空间**：正确密码进入，本地加密存储
- **公开空间**：任意错误密码进入，不加密，可正常读写使用
- 正确密码进入后可切换查看两个空间，支持将公开内容转移到私密空间
- 密码无强度限制，支持密码提示（密码输入框附近小按钮，点击显示）
- 设置中可修改密码（需验证旧密码）

#### 2.1.1 密码与加密体系

- 创建密码时，系统生成随机**主密钥（Master Key）**用于加密数据
- 密码通过 Argon2 派生出密钥A，用密钥A加密主密钥
- 恢复码为独立的随机强密码，通过 Argon2 派生出密钥B，同样加密一份主密钥
- 密码和恢复码之间无数学关联，两把锁锁同一个主密钥
- **修改密码**：用旧密码解出主密钥 → 用新密码重新加密主密钥，恢复码不变
- **恢复码重置密码**：用恢复码解出主密钥 → 设置新密码加密主密钥 → 生成新恢复码 → 旧恢复码自然失效

#### 2.1.2 恢复码

- 创建密码后显示一组恢复码，提供**复制到剪贴板**和**下载为文本文件**两个选项
- 密码登录页面有"忘记密码"链接，点击后输入恢复码重置密码
- 重置密码后会生成新的恢复码，旧恢复码失效（恢复码与密码绑定）

### 2.2 聊天式日记

- 用户消息靠右（浅绿色气泡），AI 回复靠左（白色/浅灰色气泡）
- 支持的消息类型：文字、图片、心情卡片、标签、AI 回复、Markdown 长文卡片
- **每条消息带有小时间戳**，显示记录时间
- 每条消息支持**编辑、删除、引用回复、打标签**（右键菜单操作）
- 标签可以打在单条消息上（消息级标签），也可以打在当天日记整体上（日记级标签）
- **引用回复视觉**：引用的消息显示为半透明小卡片，带圆角和柔和连接线，呈现"对话延续"的感觉
- 删除单条消息无需确认，整天删除需反复确认防止误删
- 图片插入支持选择"原图"或压缩（用户勾选）
- **图片拖拽插入**：直接从桌面/文件夹拖入聊天区即可插入，拖入时显示温暖虚线框 + "放在这里吧~" 提示，带呼吸灯效果
- 今日打开自动创建日记，全空则不保存
- 消息发出后有气泡弹出动画，整体动效活泼可爱
- **发送方式**：默认回车发送（可在设置中改为 Ctrl/Cmd+Enter 发送）
- **输入框字数统计**：输入框角落小小地显示今日已写字数

#### 2.2.1 跨天逻辑

- 用户正在当天页面输入时，不自动切换到新一天
- 新一天的空白日记自动创建好，用户可随时点击左侧列表切换
- 关闭窗口再打开、或退出软件重新进入时，自动切换到当天最新日记

### 2.3 Markdown 长文

- 工具栏"长文模式"按钮，打开**所见即所得编辑器**（Typora 风格）
- 编辑器底部显示当前文章字数统计
- 写完后作为文章卡片插入聊天流（显示标题 + 前几行预览，点击展开全文）
- 长文数量不限，可随时编辑、修改、删除
- 每篇可独立导出为 Markdown / PDF
- 当天仅有长文无聊天时，默认显示文章阅读视图，可手动切换为聊天流视图
- "文库"入口集中展示所有长文（列表形式：标题 + 日期 + 预览）

### 2.4 AI 总结与反馈

- 工具栏"AI 总结与反馈"按钮，**点击一次触发一次**（非实时对话）
- AI 阅读当天所有内容（用户消息 + AI 历史回复），给出总结性反馈
- 用户可继续写，再次点击按钮，AI 看到完整上下文再次回复
- **使用内置 AI 时每天限 10 次**，使用用户自定义 API Key 时不限次数
- AI 人格默认"温暖的朋友"，用户可在设置中自定义性格描述
- AI Provider 设置：
  - 内置 AI：MiniMax 大模型（开发者包月），开箱即用
  - 用户自定义：可配置 Provider + API Key（如 OpenAI、Claude 等），优先级高于内置

### 2.5 心情标签

- 工具栏"心情"按钮，弹出心情选择面板
- **手动选择**：直接选一个表情，插入聊天流作为心情卡片
- **AI 自动判断**：点击"AI 帮我判断"，AI 读取当天已有文字分析后生成心情卡片
- 一天可打多次心情（早上开心，晚上疲惫），也可一次都不打
- 不点击则无心情记录，完全不打扰用户

### 2.6 自定义标签

- 工具栏"标签"按钮，标签打在**当天日记整体**
- 系统预设标签：工作、生活、旅行、感悟、学习
- 用户可自定义增删标签
- **标签颜色**：每个标签自动分配柔和的莫兰迪色系颜色
- 日历视图中，有标签的日期下方显示对应颜色的小色块，一眼看出当天写了什么主题

### 2.7 Telegram / 微信远程写入

- 通过 Telegram Bot 或微信 weixin-claw 插件发消息，自动写入当天日记
- App 使用 `getUpdates` 长轮询接收 Telegram 消息
- 离线消息在 Telegram 服务器缓存最多 24 小时，App 启动后自动拉取
- 收到消息后 Bot 自动回复 ✅ 表情确认已记录
- 来自外部渠道的消息在日记中有**小角标**标识来源（Telegram / 微信图标）
- Windows 关闭 App 前，若开启了 Telegram 功能则提示用户确认

### 2.8 收藏功能

- 任何日记元素（文字、图片、长文、AI 回复、心情卡片）均可右键/长按收藏
- 独立"收藏"入口，收藏列表按时间排序，显示内容预览 + 来源日期
- 支持取消收藏
- 参考微信收藏的设计

### 2.9 快速记录模式

- 全局快捷键 `Cmd/Ctrl + Shift + M`，无论当前在使用什么软件，按下后弹出一个小浮窗
- 浮窗只有一个输入框，类似 macOS Spotlight 的极简体验
- 打完字按回车，内容自动写入今天的日记，浮窗消失
- 极大降低"打开软件才能记录"的启动成本，帮助用户随时捕捉闪念
- 适合快速记录灵感、待办、碎碎念

## 三、整体布局

### 3.1 三栏布局（参考微信 macOS 版）

```
最左侧竖栏（图标导航）  |  次级面板           |  右侧主区域
                        |                     |
  📅 日记               |  日期列表            |  聊天区
  🖼️ 相册              |  （合并为大区域）     |  照片墙 / 大图预览
  📚 文库               |  文章列表            |  文章阅读
  ⭐ 收藏               |  收藏列表            |  内容详情
  🏅 勋章               |  （合并为大区域）     |  勋章墙
  🎲 随机回忆            |                     |
                        |                     |
  ——————                |                     |
  ⚙️ 设置（左下角）      |                     |
```

- **最左侧竖栏**：固定窄宽度，只显示图标，鼠标悬浮显示文字提示
- **次级面板**：根据选中的导航项切换内容，可隐藏
- **右侧主区域**：内容展示和编辑
- **相册和勋章**：点击后次级面板与主区域合并为一个大区域，最大化利用空间
- **文库和收藏**：保持两栏（左列表 + 右详情）

### 3.2 日记列表（次级面板）

- 按天排列，显示日期 + 当天预览摘要
- 顶部搜索框，支持全文搜索
- 日历入口，独立面板可选择年/月快速跳转
- 写过日记的日期在日历上有小圆点标记
- 底部显示书写统计

### 3.3 相册

- 点击后次级面板与主区域合并，展示照片墙（网格布局）
- **单击图片**：打开大图预览（灯箱模式）
- **大图角落**：有"跳转到日记"按钮
- **右键菜单**：跳转到日记、下载、收藏
- 不直接跳转日记，默认查看大图

### 3.4 文库

- 左侧列表展示所有 Markdown 长文（标题 + 日期 + 前几行预览）
- 右侧为文章阅读区

### 3.5 收藏

- 左侧收藏列表（按时间排序，显示内容预览 + 来源日期）
- 右侧内容详情

## 四、用户粘性功能

### 4.1 书写统计（温和风格）

- 日记列表底部显示："已陪伴你 X 天，其中 Y 天留下了文字"
- 日历视图中写过日记的日期有小圆点，形成视觉"填满感"
- **不强调连续，不惩罚断更**，只是安静记录

### 4.2 年度统计报告

- 每年底自动生成统计：
  - 总篇数、总字数
  - 最常用的心情
  - 高频关键词
  - 写得最多的月份
- AI 生成一段年度总结文字
- 可导出为精美图片，方便分享

### 4.3 每日写作引导

- 空白日记页面中间显示温暖小插画（小动物等待你倾诉）+ 每日引导语
- 示例："今天有什么让你微笑的小事吗？"、"最近在读什么书？"
- 用户开始打字后引导语淡出消失
- 提示语来源：AI 生成 或 内置优质问题库
- 用户可在设置中关闭此功能

### 4.4 随机回忆

- 左侧导航栏 🎲 按钮
- 点击后随机跳转到过去某一天的日记
- 像翻旧日记本一样，带有惊喜感

### 4.5 成就勋章系统

统一在"勋章墙"页面查看所有已获得和未获得的勋章。获得勋章时弹出温暖的庆祝小动画。

## 五、系统功能

### 5.1 数据存储

- 纯本地存储，不依赖服务器
- 私密空间：SQLite + AES-256-GCM 加密，密码派生使用 Argon2
- 公开空间：SQLite，不加密
- **图片存储**：采用折中方案
  - 缩略图（压缩后几十KB）存入数据库
  - 原图存本地专属文件夹，数据库中存相对路径
  - 原图文件做 AES 加密存储，文件名使用哈希，直接打开是乱码
  - 导出时自动将数据库 + 加密图片文件夹一起打包为压缩包
  - 导入时自动解压还原，用户无感

### 5.2 数据迁移

- **导出**：将数据库 + 图片文件夹自动压缩为压缩包
- **导入**：支持两种模式
  - **全新导入**：选择压缩包 → 输入该数据库密码验证 → 通过后导入（覆盖当前数据）
  - **合并导入**：选择压缩包 → 输入该数据库密码验证 → 通过后与当前数据合并
  - 如果两个数据库密码不同，需要输入外来数据库的密码
- **删除所有数据**：设置中提供一键删除选项，需反复确认，删除后恢复为全新软件状态
- 首次使用时引导用户选择"创建全新日记本"或"从备份文件导入"

### 5.3 导出功能

- 整库导出/导入（压缩包）
- 单篇日记导出，用户可选择：
  - **气泡截图风格**：保留聊天气泡样式（类似微信聊天记录截图）
  - **排版文档风格**：转换为传统日记排版格式
  - 导出格式：PDF / 图片
- 长文独立导出（Markdown / PDF）

### 5.4 多语言

- 支持中文和英文
- 自动检测系统语言：中文系统显示中文，其他语言显示英文
- 设置中可手动切换语言

### 5.5 字体与显示

- 字体大小可在设置中自定义
- 支持 `Cmd/Ctrl + =` 放大、`Cmd/Ctrl + -` 缩小整个 UI
- 屏幕大小自动适配（使用 pretext 开源库）

### 5.6 窗口行为

- **macOS**：点击关闭按钮隐藏窗口，不退出应用
- **Windows**：点击关闭按钮退出应用；若开启了 Telegram 功能，关闭前弹窗提示确认

### 5.7 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Cmd/Ctrl + F` | 搜索 |
| `Cmd/Ctrl + T` | 切换到今天 |
| `Cmd/Ctrl + ←` | 前一天 |
| `Cmd/Ctrl + →` | 后一天 |
| `Cmd/Ctrl + E` | 打开长文编辑器 |
| `Cmd/Ctrl + =` | 放大 UI |
| `Cmd/Ctrl + -` | 缩小 UI |
| `Cmd/Ctrl + Shift + M` | 快速记录（全局快捷键，弹出浮窗） |
| `Esc` | 关闭大图/面板 |

### 5.8 搜索

- 全文搜索范围：聊天文字消息、长文内容、AI 回复内容、标签名称、心情标签
- 搜索结果高亮匹配文字，点击可跳转到对应日期

### 5.9 定时提醒

- 默认关闭
- 用户可在设置中开启，自定义提醒时间
- 开启后在设定时间弹出系统通知提醒写日记

### 5.10 更新机制

- 应用商店版本使用商店自带更新机制
- 设置中有"检查更新"按钮，用户主动点击时检查
- 不主动弹出更新提示

## 八、用户流程

### 8.1 首次使用

```
打开 App
  → 欢迎动画 + 产品简介（可跳过）
  → 选择：「创建全新日记本」or「从备份文件导入」

  创建全新：
    → 设置密码（输入两次确认）
    → 设置密码提示语（可选，可跳过）
    → 显示恢复码（提供复制 + 下载为文本文件）
    → 进入主界面，今日空白日记

  从备份导入：
    → 选择压缩包文件
    → 输入该数据库的密码（验证通过才可导入）
    → 进入主界面
```

### 8.2 日常使用

```
打开 App
  → 密码输入界面（温暖问候语 + 动效，如"又见面了"、"今天过得怎么样？"）
  → 输入密码

  正确密码 → 进入私密空间（可切换到公开空间）
  错误密码 → 进入公开空间（用户无感知）

  → 默认显示今天的日记
  → 写文字、发图片、插入长文、打心情、打标签
  → 需要 AI 反馈时点击按钮触发
```

### 8.3 密码忘记流程

```
密码输入界面 → 点击"忘记密码"
  → 输入恢复码
  → 验证通过 → 设置新密码（输入两次确认）
  → 生成新的恢复码（提供复制 + 下载）
  → 旧恢复码失效
  → 进入主界面
```

## 九、UI 风格与体验

### 9.1 视觉风格

- **整体风格**：浅色、温暖、柔和，参考微信 macOS 版美术风格
- **关键词**：简约、温柔、温暖、轻松、有倾诉欲望
- **气泡颜色**：用户消息浅绿色（靠右），AI 回复白色/浅灰色（靠左）
- 圆角、充足留白、柔和阴影

### 9.2 动效与微交互

- **消息发送**：气泡弹出动画，活泼可爱
- **勋章获得**：温暖的庆祝小动画弹出
- **密码登录界面**：温馨动效 + 随机温暖问候语
- **心情选择面板**：表情弹出动画
- **页面切换**：柔和过渡动画
- **输入框呼吸感**：用户打字时输入框边框有极淡的柔光脉动效果，像日记本在"倾听"你说话；停止打字几秒后脉动缓缓消失
- **图片拖入反馈**：拖入图片时聊天区出现温暖虚线框 + 呼吸灯效果
- **导出动画**：单篇日记导出时有封蜡章按压的小动画
- 整体动效风格统一：活泼、可爱、不突兀

### 9.3 时间氛围背景

- 聊天区背景随时间段有微妙的氛围色调变化：
  - 早晨（6:00-11:00）：暖黄色微光
  - 下午（11:00-17:00）：明亮柔白
  - 傍晚（17:00-20:00）：淡橘色
  - 深夜（20:00-6:00）：深蓝色调，稍暗
- 变化非常轻微，不影响阅读，无意识地营造"时间在流淌"的感觉
- 回看历史日记时，背景还原为当时的时间氛围
- 用户可在设置中关闭（改为固定背景）

### 9.4 季节粒子动效

- 随季节在聊天区有极其轻微的粒子效果：
  - 春天：偶尔飘过淡淡的樱花瓣
  - 夏天：聊天区底部微弱的水波纹
  - 秋天：偶尔飘落一两片枫叶
  - 冬天：角落有轻微的雪花粒子
- 粒子非常稀疏、缓慢、半透明，不干扰阅读
- 用户可在设置中关闭

### 9.5 特殊日期彩蛋

- **用户生日**（如在设置中填写了）：当天打开时有生日蛋糕动画 + "生日快乐，记录美好的一年"
- **新年元旦**：打开时有烟花动画
- **使用周年纪念日**："你已经和喃喃在一起整整一年了" + 庆祝动画
- 这些都是一次性小惊喜，当天只出现一次

### 9.6 日记本成长树

- 勋章页面背景展示一棵成长树
- 使用时间越长，树从种子 → 小苗 → 小树 → 大树逐渐茂盛
- 获得的勋章挂在树枝上
- 树的当前状态也显示在勋章页面："这本日记本已经 128 天了"

### 9.7 空状态设计

- **空白日记（今日未写）**：中间显示温暖小插画（小动物坐着等你倾诉）+ 每日引导语，用户开始打字后淡出
- **相册为空**：温馨插画 + 提示语（如"还没有图片呢，记录生活的美好瞬间吧"）
- **文库为空**：温馨插画 + 提示语（如"写下你的第一篇长文吧"）
- **收藏为空**：温馨插画 + 提示语（如"收藏你珍视的每一段文字"）
